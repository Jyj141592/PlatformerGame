# 개요

PlayerController 테스용 프로젝트(에디터 버전 2022.3.18f1)

Jump King과 유사한 방식으로 점프를 통해 떨어지지 않고 위쪽으로 올라가는 게임

# 조작법

좌우 화살표로 이동, 스페이스바를 눌러 점프. 스페이스바를 누르고 있는 시간에 따라 점프의 각도가 0~90도로 증가
